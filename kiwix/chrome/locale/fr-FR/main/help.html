Bienvenue
Bienvenue dans Kiwix. Kiwix est un logiciel libre capable de lire les fichiers ZIM. Les fichiers ZIM sont étudiés pour compresser dans un minimum d'espace des contenus multimédias (text, image ou vidéo), typiquement issus du World Wide Web, comme par exemple de Blogs ou encore de Wikis. Kiwix est donc une solution si vous souhaitez pouvoir consulter hors ligne des contenus qui sont normalement uniquement disponibles en ligne. Il a été en particulier développé pour donner accès aux contenus de l'encyclopédie libre Wikipédia aux personnes ne disposant pas d'un accès aisé à Internet. Bien que Kiwix soit un logiciel en perpétuel développement il est déjà utilisable.
Fichier ZIM
Il n'existe pas encore de fichiers ZIM vraiment terminés, mais nous travaillons sur quelques uns. Vous pouvez télécharger les versions de développement ici avec votre navigateur Web et une fois le téléchargement terminé l'ouvrir avec le menu "Fichier > Ouvrir un fichier..."
